Tiles, Maps, Terrain, Oh My!

I have interest in building virtual worlds that can be used in games or apps. There was an interesting demo by @wooorldXR that really inspired me into trying my hand in learning to build a 3D map viewer from scratch. So this repo will contain all the little experiements & prototypes using OSM data to recreate that demo.

Ideally this will be a great place for people to learn how to do it themselves by reviewing all the feature focused prototypes instead of digging into large projects like cesium. Focused bare bones non-optimized code is really the best sort of code to learn from. From there you can build something better for production.

Prototypes

  3. Build a multi-part mesh from OSM Building (Empire State Building)
  4. Create 3D Terrain tile from Mapzen's terrarium & normal map tiles along with ArcGIS satellite map tile
